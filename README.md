C语言运算符学习日志 2026/7/20
 
一、今日学习内容
 
本次在Operators项目工程下分文件练习各类运算符，拆分多个 .c 文件模块化练习，覆盖四类核心运算符：
 
1. 算术/计算运算符（Operators.c）
基础四则运算、取模运算，熟悉整数除法向下取整规则，区分浮点与整型运算差异。
2. 递增递减运算符 ++ / --（increment.c）
区分前置 ++i 、后置 i++ 的运算顺序；前置先自增再取值，后置先取值再自增，结合printf打印直观对比两者区别。
3. 逻辑判断运算符（logic_judgement.c、condition.c）
逻辑与 && 、逻辑或 || 、逻辑非 ! ；掌握短路求值特性；练习三目条件运算符  条件 ? 真表达式 : 假表达式 ，完成分数等级判断案例。
4. 按位/位运算符（bitwise.c、shift.c）
按位与 & 、按位或 | 、按位异或 ^ 、按位取反 ~ ；左移 << 、右移 >> 移位运算，理解二进制底层运算逻辑。
 
二、项目文件结构梳理
 
-  .c 源文件：分题型单独拆分，单一文件只写一类运算符代码，结构清晰便于复习
- Operators.c：基础计算运算符
- increment.c：自增自减
- logic_judgement.c：基础逻辑运算符
- condition.c：三目运算符条件判断
- bitwise.c：基础按位运算
- shift.c：左右移位运算
- daycode.h：自定义头文件，可存放复用常量、函数声明
- Operators.sln / Operators.vcxproj：VS解决方案与工程配置文件，统一管理全部练习代码
- .vs、x64：编译生成缓存与程序输出目录，提交git时可配置.gitignore忽略
