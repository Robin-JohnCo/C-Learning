# C语言整型学习日志
## 项目介绍
本人C语言入门练习仓库，专题：整型数据完整演示。
整个工程仅存在唯一main入口，各示例拆分独立函数，按需运行。

## 文件结构
- ConsoleApplication1/Robin.c：程序唯一入口main函数
- ConsoleApplication1/src/Robin_2.c：size,sizeof演示1
- ConsoleApplication1/src/Robin_3.c：size,sizeof演示2
- ConsoleApplication1/src/Robin_4.c：显式转换和隐式转换
- ConsoleApplication1/src/Robin_5.c：企业化表达
- ConsoleApplication1/src/Robin_6.c：scanf_s初步运用

## 使用方法
1. 使用Visual Studio打开 ConsoleApplication1.sln
2. 在Robin.c的main函数内，取消对应示例函数注释即可运行
3. 所有模块无独立main，不会多重定义报错
